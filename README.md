## Installation & Setup

1. Upload the plugin zip file through WordPress plugins page
2. Activate the plugin
3. Go to "AI Reviews Settings" page
4. Enter your OpenRouter API key
5. Select preferred AI model
6. Apply your desired settings

## Required Configuration

### API Key Setup
1. Visit [OpenRouter](https://openrouter.ai/) website
2. Create a user account
3. Generate a new API key from API Keys section
4. Enter the key in plugin settings

### Time Settings
- Set automatic generation time window
- Configure number of posts processed per run
- Specify number of comments per post

## How to Use

### Automatic Generation
The plugin automatically generates comments during night hours. Configure night hours in settings page.

### Manual Generation
1. Go to "Reviews List" page
2. Click "Manual Review Generation" button
3. Wait for the process to complete

### Review Management
1. View generated comments in "Reviews List" page
2. Click "Edit" to modify comments
3. Click "Approve & Publish" to publish comments
