## معرفی افزونه

افزونه تولید خودکار نظرات هوش مصنوعی، یک راهکار هوشمند برای تولید محتوای تعاملی در سایت‌های وردپرس است. این افزونه با استفاده از هوش مصنوعی OpenRouter، به صورت خودکار نظرات طبیعی و مرتبط برای مطالب و محصولات سایت تولید می‌کند.

## ویژگی‌های کلیدی

- 🔥 **تولید خودکار نظرات**: تولید نظرات به صورت خودکار در ساعات مشخص شده
- 🤖 **پشتیبانی از چندین مدل AI**: استفاده از مدل‌های مختلف هوش مصنوعی OpenRouter
- ⚡ **انتشار هوشمند**: انتشار خودکار نظرات در سیستم نظرات وردپرس
- ✏️ **ویرایش آسان**: امکان ویرایش نظرات قبل از انتشار
- 📊 **مدیریت پیشرفته**: صفحه مدیریت اختصاصی برای نظارت بر نظرات تولید شده
- 🔧 **تنظیمات قابل سفارشی‌سازی**: کنترل کامل بر رفتار افزونه

## امکانات افزونه

### تولید هوشمند نظرات
- تولید نظرات طبیعی و خوانا به زبان فارسی
- استفاده از هوش مصنوعی برای ایجاد نظرات مرتبط با محتوا
- قابلیت تنظیم تعداد نظرات تولیدی برای هر مطلب

### مدیریت زمان اجرا
- اجرای خودکار در ساعات شب (قابل تنظیم)
- وقفه‌های زمانی برای جلوگیری از overload
- اجرای دستی در هر زمان دلخواه

### مدیریت نظرات
- پیش‌نمایش کامل نظرات تولید شده
- امکان ویرایش نام نویسنده و متن نظر
- تایید و انتشار یک‌جا یا تکی نظرات
- حذف نظرات ناخواسته

### یکپارچه‌سازی با وردپرس
- ذخیره نظرات در دیتابیس وردپرس
- انتشار خودکار در سیستم نظرات وردپرس
- سازگاری کامل با انواع پست‌ها و محصولات

## نصب و راه‌اندازی

1. فایل زیپ افزونه را از طریق صفحه افزونه‌ها در وردپرس آپلود کنید
2. افزونه را فعال نمایید
3. به صفحه «تنظیمات نظرات AI» مراجعه کنید
4. کلید API OpenRouter خود را وارد نمایید
5. مدل هوش مصنوعی مورد نظر را انتخاب کنید
6. تنظیمات دلخواه خود را اعمال نمایید

## تنظیمات لازم

### دریافت کلید API
1. به سایت [OpenRouter](https://openrouter.ai/) مراجعه کنید
2. حساب کاربری ایجاد نمایید
3. از بخش API Keys، یک کلید جدید ایجاد کنید
4. کلید را در تنظیمات افزونه وارد نمایید

### تنظیمات زمانی
- تعیین بازه زمانی تولید خودکار نظرات
- تنظیم تعداد مطالب پردازش شده در هر اجرا
- مشخص کردن تعداد نظرات برای هر مطلب

## نحوه استفاده

### تولید خودکار
افزونه به صورت خودکار در ساعات شب نظرات تولید می‌کند. برای تنظیم ساعت شب به صفحه تنظیمات مراجعه کنید.

### تولید دستی
1. به صفحه «لیست نظرات» مراجعه کنید
2. روی دکمه «اجرای دستی تولید نظرات» کلیک کنید
3. منتظر بمانید تا فرآیند تولید کامل شود

### مدیریت نظرات
1. در صفحه «لیست نظرات» می‌توانید نظرات تولید شده را مشاهده کنید
2. با کلیک روی «ویرایش» می‌توانید نظرات را تغییر دهید
3. با کلیک روی «تایید و انتشار» نظرات را منتشر کنید

## پشتیبانی و راهنمایی

برای راهنمایی بیشتر می‌توانید از صفحات زیر استفاده کنید:
- صفحه «راهنمای استفاده» برای آموزش کامل
- صفحه «وضعیت سیستم» برای بررسی عملکرد افزونه

## امنیت و حریم خصوصی

- ارتباطات با سرورهای OpenRouter به صورت SSL加密 شده است
- کلیه عملیات با مجوزهای مناسب انجام می‌شود

## سیستم مورد نیاز

- وردپرس 5.0 یا بالاتر
- PHP 7.4 یا بالاتر
- دسترسی به API خارجی (OpenRouter)
- فعال بودن curl در سرور

## Introduction

AI Auto Review Generator is a smart solution for creating interactive content on WordPress sites. This plugin uses OpenRouter AI to automatically generate natural and relevant comments for posts and products.

## Key Features

- 🔥 **Auto Review Generation**: Automatic comment generation during specified hours
- 🤖 **Multiple AI Models Support**: Use various OpenRouter AI models
- ⚡ **Smart Publishing**: Automatic publishing to WordPress comment system
- ✏️ **Easy Editing**: Edit comments before publishing
- 📊 **Advanced Management**: Dedicated admin panel for review management
- 🔧 **Customizable Settings**: Full control over plugin behavior

## Plugin Capabilities

### Smart Review Generation
- Natural and readable Persian language comments
- AI-powered context-aware comment generation
- Configurable number of comments per post

### Scheduling Management
- Automatic night-time execution (configurable)
- Time intervals to prevent overload
- Manual execution at any time

### Review Management
- Full preview of generated comments
- Author name and comment text editing
- Bulk or individual approval and publishing
- Unwanted comment removal

### WordPress Integration
- Storage in WordPress database
- Automatic publishing to WordPress comment system
- Full compatibility with posts and products

## Installation & Setup

1. Upload the plugin zip file through WordPress plugins page
2. Activate the plugin
3. Go to "AI Reviews Settings" page
4. Enter your OpenRouter API key
5. Select preferred AI model
6. Apply your desired settings

## Required Configuration

### API Key Setup
1. Visit [OpenRouter](https://openrouter.ai/) website
2. Create a user account
3. Generate a new API key from API Keys section
4. Enter the key in plugin settings

### Time Settings
- Set automatic generation time window
- Configure number of posts processed per run
- Specify number of comments per post

## How to Use

### Automatic Generation
The plugin automatically generates comments during night hours. Configure night hours in settings page.

### Manual Generation
1. Go to "Reviews List" page
2. Click "Manual Review Generation" button
3. Wait for the process to complete

### Review Management
1. View generated comments in "Reviews List" page
2. Click "Edit" to modify comments
3. Click "Approve & Publish" to publish comments
